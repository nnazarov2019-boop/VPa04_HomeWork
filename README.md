# Домашняя работа к уроку VPa04
---

## Содержание задания
Составление промптов и предоставление отчёта по шаблону, предложенному в уроке.
***

## Порядок выполнения
Составляются три промпта: плохой, средний и хороший.
По хорошему промпту составляется программа.
При необходимости производтися исправление ошибок и коррекция.
Промты и отлаженная программа выкладываются в GitHub. Ссылка на репозиторий отправляется в качестве отчёта по ДЭ.
