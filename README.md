## Отчёт о проделанной работе

### Запуск программы

Запускать программу можно двумя способами:
- без аргументов;
- с аргументами.

В варианте без аргументов программа сама предложит ввести порядок числителя и знаменателя. Формат запуска программы:

```bash
python src/main.py
```

В варианте с аргументами пользователь может указать аргументы прямо в командной строке. Формат запуска программы:

```bash
python src/main.py <порядок числителя> <порядок знаменателя>
```

Пример:

```bash
python src/main.py 2 4
```
***

### Порядок проверки работоспособности
Работоспособность можно проверять автоматически путём запуска программы проверки. Формат запуска программы:

```bash
python tests/run_tests.py
```

Программа запускает 12 тестов, каждый из которых проверяет реакцию `main.py` на различные входные данные (корректные, ошибочные, граничные случаи). Результат каждого теста выводится в формате:

```
ТЕСТ №1: Корректный вызов: m=2, n=4
  → НОРМА
ТЕСТ №3: Ошибка: m > n (m=5, n=3)
  → НОРМА
...
============================================================
  ИТОГ ТЕСТИРОВАНИЯ
============================================================
  Всего тестов: 12
  Пройдено:     12
  Провалено:    0
============================================================
```
