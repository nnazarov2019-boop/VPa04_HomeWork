Напиши на Python консольное приложение для преобразования передаточной функции в описание в пространстве состояний. 
Особенности реализации:
- все преобразования должны быть символическими;
- процедура преобразования должна быть оформлена в виде функции;
- приложение должно быть консольным;
- все входные данные пользователь должен вводить в консоли;
- все выходные данные должны выводиться в консоль.
- коэффициенты числителя передаточной функции должны обозначаться b с индексом. Коэффициенты знаменателя должны обозначаться a с индексом.

Функция преобразования передаточной функции в описание в пространстве состояний должно удовлетворять следующим требованиям:
Входные данные: 
- порядок числителя (задаётся натуральным числом);
- порядок знаменателя (задается натуральным числом).

Выходные данные: 
- матрицы пространства состояний.
